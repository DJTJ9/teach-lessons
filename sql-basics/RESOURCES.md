# Resources: SQL Basics

## Knowledge
- [PostgreSQL Docs](https://www.postgresql.org/docs/current/) — offizielle Referenz, sehr vollstaendig
- [SQLZoo](https://sqlzoo.net/) — interaktive SQL-Uebungen, gut fuer Einstieg
- [pgexercises.com](https://pgexercises.com/) — PostgreSQL-spezifische Uebungen mit sofortigem Feedback

## Wisdom (communities)
- [r/PostgreSQL](https://reddit.com/r/PostgreSQL) — aktive Community fuer PostgreSQL-Fragen
- [Database Administrators Stack Exchange](https://dba.stackexchange.com/) — hochqualitative Antworten
