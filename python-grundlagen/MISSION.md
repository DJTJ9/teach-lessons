# Mission

## Why
Python so beherrschen, dass eigenständig AI-Engineering-Projekte und Apps gebaut werden können — beginnend mit einem CLI-Chatbot, der eine LLM-API nutzt und Notion anbindet.

## Success looks like
- Eigenen CLI-Chatbot in Python gebaut: nimmt Freitexteingaben, ruft LLM-API auf, schreibt/liest Notion-Daten
- bot.py vollständig verstanden: jede Funktion, jedes Pattern erklären können
- Typische Python-Patterns für AI Engineering anwenden: Prompt Engineering, API-Calls, JSON-Handling, Threading
- Pytest-Tests für eigenen Code schreiben können

## Constraints
- Starkes C#-Vorwissen — Python immer im C#-Vergleich lernen
- Kein Zeitlimit, Qualität über Tempo
- Kein Framework-Overhead: erst pure Python, dann nur was gebraucht wird
- Lektionen werden gesammelt, dann eigenständig durchgearbeitet

## Out of scope
- Web-Frameworks (FastAPI, Django) — erst nach Abschluss dieses Kurses
- Data Science / Pandas / NumPy — eigener Kurs bei Bedarf
- Packaging / PyPI-Publishing
- Async/await — Vertiefung nach den Grundlagen
