# Resources

## Knowledge
- [Python Docs](https://docs.python.org/3/) - Offizielle Referenz; bei Fragen zu Stdlib immer zuerst hier
- [Real Python](https://realpython.com/) - Tiefe, praxisnahe Artikel; gut für "wie macht man X in Python richtig"
- [Groq API Docs](https://console.groq.com/docs) - LLM-API die bot.py nutzt; für LLM-Lektionen
- [Anthropic API Docs](https://docs.anthropic.com/) - Claude API; Referenz für eigene AI-Projekte
- [Notion API Docs](https://developers.notion.com/) - Notion-Datenbankoperationen; bot.py nutzt MCP-Layer darüber

## Wisdom (communities)
- [r/learnpython](https://reddit.com/r/learnpython) - Anfänger-freundlich, deutsche Posts OK
- [Python Discord](https://discord.gg/python) - Aktiv, schnelle Antworten
