Ziel: Alle gelernten Java-Konzepte in einer vollständigen, funktionsfähigen Anwendung zusammenführen.

Was bauen wir?

Eine Konsolen-App die Todos verwaltet:

  • Todo hinzufügen
  • Alle Todos anzeigen
  • Todo als erledigt markieren
  • Todos in Datei speichern und laden

Projektstruktur

todo-app/
  src/main/java/com/todo/
    Todo.java          ← Datenklasse
    TodoManager.java   ← Logik
    App.java           ← Einstiegspunkt + UI

Schritt 1 — Todo-Klasse

Todo.java

package com.todo;

public class Todo {
    private String text;
    private boolean erledigt;

    public Todo(String text) {
        this.text = text;
        this.erledigt = false;
    }

    public String getText() { return text; }
    public boolean isErledigt() { return erledigt; }
    public void setErledigt(boolean erledigt) { this.erledigt = erledigt; }

    @Override
    public String toString() {
        String status = erledigt ? "[x]" : "[ ]";
        return status + " " + text;
    }
}

Schritt 2 — TodoManager

TodoManager.java

package com.todo;

import java.util.ArrayList;
import java.util.List;
import java.nio.file.Files;
import java.nio.file.Path;
import java.io.IOException;

public class TodoManager {
    private ArrayList<Todo> todos = new ArrayList<>();
    private static final Path DATEI = Path.of("todos.txt");

    public void hinzufuegen(String text) {
        todos.add(new Todo(text));
    }

    public List<Todo> alleTodos() {
        return todos;
    }

    public void alsErledigt(int index) {
        if (index >= 0 && index < todos.size()) {
            todos.get(index).setErledigt(true);
        } else {
            throw new IllegalArgumentException("Ungültiger Index: " + index);
        }
    }

    public void speichern() throws IOException {
        ArrayList<String> zeilen = new ArrayList<>();
        for (Todo todo : todos) {
            zeilen.add((todo.isErledigt() ? "1" : "0") + ":" + todo.getText());
        }
        Files.write(DATEI, zeilen);
    }

    public void laden() throws IOException {
        if (!Files.exists(DATEI)) return;
        List<String> zeilen = Files.readAllLines(DATEI);
        for (String zeile : zeilen) {
            String[] teile = zeile.split(":", 2);
            Todo todo = new Todo(teile[1]);
            if (teile[0].equals("1")) todo.setErledigt(true);
            todos.add(todo);
        }
    }
}

Schritt 3 — App (Einstiegspunkt)

App.java

package com.todo;

import java.util.Scanner;
import java.io.IOException;

public class App {
    public static void main(String[] args) {
        TodoManager manager = new TodoManager();
        Scanner scanner = new Scanner(System.in);

        try {
            manager.laden();
            System.out.println("Todos geladen.");
        } catch (IOException e) {
            System.out.println("Keine gespeicherten Todos gefunden.");
        }

        while (true) {
            System.out.println("\n--- TODO APP ---");
            System.out.println("1) Alle Todos anzeigen");
            System.out.println("2) Todo hinzufügen");
            System.out.println("3) Todo erledigen");
            System.out.println("4) Speichern und beenden");
            System.out.print("Auswahl: ");

            String eingabe = scanner.nextLine().trim();

            switch (eingabe) {
                case "1":
                    var todos = manager.alleTodos();
                    if (todos.isEmpty()) {
                        System.out.println("Keine Todos vorhanden.");
                    } else {
                        for (int i = 0; i < todos.size(); i++) {
                            System.out.println((i + 1) + ". " + todos.get(i));
                        }
                    }
                    break;

                case "2":
                    System.out.print("Neues Todo: ");
                    String text = scanner.nextLine().trim();
                    if (!text.isEmpty()) {
                        manager.hinzufuegen(text);
                        System.out.println("Todo hinzugefügt!");
                    }
                    break;

                case "3":
                    System.out.print("Welche Nummer erledigen? ");
                    try {
                        int num = Integer.parseInt(scanner.nextLine().trim()) - 1;
                        manager.alsErledigt(num);
                        System.out.println("Erledigt!");
                    } catch (NumberFormatException e) {
                        System.out.println("Keine gültige Nummer.");
                    } catch (IllegalArgumentException e) {
                        System.out.println(e.getMessage());
                    }
                    break;

                case "4":
                    try {
                        manager.speichern();
                        System.out.println("Gespeichert. Tschüss!");
                    } catch (IOException e) {
                        System.out.println("Fehler beim Speichern: " + e.getMessage());
                    }
                    scanner.close();
                    return;

                default:
                    System.out.println("Ungültige Eingabe.");
            }
        }
    }
}

🧠 Erweiterungsaufgaben

Versuche diese Erweiterungen selbst umzusetzen:

1. Todo löschen hinzufügen
Methode loeschen(int index) in TodoManager: todos.remove(index). Dann case "5" in App.
2. Nur offene Todos anzeigen
In alleTodos() filtern: if (!todo.isErledigt()) in einer neuen Methode offeneTodos().
3. Priorität zu Todo hinzufügen (Hoch/Mittel/Niedrig)
Neues Enum Prioritaet { HOCH, MITTEL, NIEDRIG }, Feld in Todo, Konstruktor anpassen, Speicher/Laden aktualisieren.
