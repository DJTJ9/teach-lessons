☕ Lektion 05 — Collections: ArrayList & HashMap

Ziel: Die wichtigsten Java-Collections verstehen und nutzen — ArrayList (= List<T>) und HashMap (= Dictionary<K,V>).

ArrayList — Javas List<T>

☕ Java — ArrayList

import java.util.ArrayList;

ArrayList<String> namen = new ArrayList<>();
namen.add("Anna");
namen.add("Ben");
namen.add("Clara");

// Lesen:
String erster = namen.get(0);   // "Anna"
int groesse = namen.size();      // 3

// Enthält:
boolean hat = namen.contains("Ben");   // true

// Entfernen:
namen.remove("Ben");
namen.remove(0);  // nach Index

// Iteration:
for (String name : namen) {
    System.out.println(name);
}

💜 C# — List<T>

using System.Collections.Generic;

List<string> namen = new List<string>();
namen.Add("Anna");
namen.Add("Ben");
namen.Add("Clara");

// Lesen:
string erster = namen[0];     // "Anna"
int groesse = namen.Count;    // 3

// Enthält:
bool hat = namen.Contains("Ben");  // true

// Entfernen:
namen.Remove("Ben");
namen.RemoveAt(0);  // nach Index

// Iteration:
foreach (string name in namen) {
    Console.WriteLine(name);
}

HashMap — Javas Dictionary<K,V>

☕ Java — HashMap

import java.util.HashMap;

HashMap<String, Integer> punkte = new HashMap<>();
punkte.put("Anna", 95);
punkte.put("Ben", 87);
punkte.put("Clara", 92);

// Lesen:
int annaPunkte = punkte.get("Anna");   // 95

// Enthält:
boolean hatKey = punkte.containsKey("Ben");   // true
boolean hatVal = punkte.containsValue(87);    // true

// Entfernen:
punkte.remove("Ben");

// Alle Keys/Values:
for (String key : punkte.keySet()) {
    System.out.println(key + ": " + punkte.get(key));
}

// Key nicht vorhanden:
Integer wert = punkte.get("Unbekannt");  // null!

💜 C# — Dictionary

var punkte = new Dictionary<string, int>();
punkte["Anna"] = 95;
punkte["Ben"] = 87;
punkte["Clara"] = 92;

// Lesen:
int annaPunkte = punkte["Anna"];   // 95

// Enthält:
bool hatKey = punkte.ContainsKey("Ben");
bool hatVal = punkte.ContainsValue(87);

// Entfernen:
punkte.Remove("Ben");

// Alle Keys/Values:
foreach (var kv in punkte) {
    Console.WriteLine($"{kv.Key}: {kv.Value}");
}

// Sicheres Lesen:
if (punkte.TryGetValue("x", out int val)) { }

Sicheres Lesen aus HashMap

// HashMap.get() gibt null zurück wenn Key nicht existiert!
// Das kann zu NullPointerException führen:
int wert = punkte.get("NichtVorhanden");  // NullPointerException!

// Sicherer — getOrDefault():
int wert = punkte.getOrDefault("NichtVorhanden", 0);  // 0 als Fallback

// Prüfen vor Zugriff:
if (punkte.containsKey("Anna")) {
    int p = punkte.get("Anna");
}

🧠 Quiz

Frage 1: Was gibt hashMap.get("NichtVorhanden") zurück wenn der Key fehlt?

A) Eine Exception
❌ Nicht direkt. get() gibt null zurück. Die Exception kommt erst wenn du null auto-unboxst (z.B. in int).
B) null
✅ Richtig! Nutze getOrDefault(key, fallback) für sicheren Zugriff.

Frage 2: Wie liest man das Element an Index 2 aus einer Java-ArrayList?

A) liste[2]
❌ Das ist C#. In Java: liste.get(2)
B) liste.get(2)
✅ Richtig! ArrayList hat keine Index-Operator-Syntax wie C#.

Aufgabe: Erstelle eine HashMap die Ländernamen (String) auf Einwohnerzahlen (Integer) mappt. Füge 3 Länder ein und gib alle aus.

Zeig mir die Lösung
HashMap<String, Integer> laender = new HashMap<>();
laender.put("Deutschland", 84000000);
laender.put("Österreich", 9000000);
laender.put("Schweiz", 8700000);

for (String land : laender.keySet()) {
    System.out.println(land + ": " + laender.get(land));
}

📋 Zusammenfassung

  • ArrayList<T> = C# List<T> — dynamische Liste
  • list.get(i) statt list[i], list.size() statt list.Count
  • HashMap<K,V> = C# Dictionary<K,V>
  • map.put(key, value) statt dict[key] = value
  • map.get(key) gibt null bei fehlendem Key — getOrDefault() für Sicherheit
  • Alle Methoden: lowercase in Java, PascalCase in C#
