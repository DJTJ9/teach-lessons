# Mission

## Why
Java steht in vielen Jobbeschreibungen als Grundvoraussetzung. Ziel: Java-Code lesen, verstehen und einfache Logik selbst umsetzen auf Junior-Niveau.

## Success looks like
- Java-Code lesen und verstehen (Klassen, Methoden, Collections)
- Unterschiede zwischen Java und C# erklären können
- Maven-Projekt aufsetzen und ausführen
- Einfache Konsolenanwendungen in Java schreiben
- Todo-App als Praxisprojekt fertigstellen

## Constraints
- Vorwissen: C# (aus Unity-Spieleentwicklung) — Vergleiche als Hauptlernmethode
- Kein Java-Vorwissen
- 6+ Stunden pro Woche
- Java 21 LTS als Zielversion

## Out of scope
- Spring Boot / Jakarta EE / Frameworks
- Android-Entwicklung
- Multithreading / Concurrency
- Java-spezifische Design Patterns
- Generics im Detail
