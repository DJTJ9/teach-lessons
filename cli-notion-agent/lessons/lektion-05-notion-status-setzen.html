Lektion 05 — Notion-Status aktualisieren

Kurs: CLI × Notion × Coding Agent  |  Heute: Properties schreiben

🎯 Lernziel

Du kannst den Status eines Notion-Tasks per Claude Code von „Not started" auf „Done" setzen — ohne Browser.

Warum ist das wichtig?

Lesen ist die halbe Miete. Euer Workflow braucht aber den Rückkanal: Claude Code erledigt eine Aufgabe → Notion weiß es → du siehst es sofort in der App. Das ist der Schritt von „Agent liest Tasks" zu „Agent schreibt Ergebnisse zurück".

Das Tool: notion_update_page_properties

Konzept

Jede Notion-Page hat eine eindeutige page_id. Properties werden als JSON-Objekt übergeben.
Status-Properties brauchen ein spezielles Format: { "status": { "name": "Done" } }

Der Ablauf

1. notion_find → 2. notion_query_data_source → 3. page_id merken → 4. notion_update_page_properties

Schritt für Schritt

1. Tasks laden (kennst du schon)

// notion_find → data_source_id ermitteln
// notion_query_data_source → Einträge + page_ids anzeigen

2. page_id identifizieren

In der Query-Antwort hat jede Page eine id im Format xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx. Das ist deine page_id.

3. Status aktualisieren

⚠ Achtung: Status-Namen sind exakt

Der Name muss exakt mit dem Notion-Status übereinstimmen (Groß-/Kleinschreibung, Leerzeichen). Falsche Namen → Notion-Fehler. Nutze notion_inspect_data_source um erlaubte Werte zu sehen.

// Tool: notion_update_page_properties
{
  "page_id": "37a4bba2-9c55-8039-ae03-e774679e0a9d",  // z.B. "Learn Notion"
  "properties": {
    "Status": {
      "status": {
        "name": "Done"
      }
    }
  }
}

Property-Typen Übersicht

TypFormat im JSONBeispiel
status{"status": {"name": "Done"}}Workflow-Status
title{"title": [{"text": {"content": "Neu"}}]}Seitentitel
rich_text{"rich_text": [{"text": {"content": "Text"}}]}Notizfeld
checkbox{"checkbox": true}Erledigt-Haken
select{"select": {"name": "Option"}}Einzelauswahl

🔬 Aufgabe — Mach es live

Live-Aufgabe

Sage Claude Code (in diesem Chat-Fenster):

„Setze den Status des Tasks 'Learn Notion' in der Tasks-Datenbank auf 'Done'."

Claude Code führt die nötigen Notion-MCP-Calls aus. Prüfe danach in der Notion Desktop App, ob der Status sich geändert hat.
