Lektion 14 — Lernthemen-Sammlung in Notion anlegen

Kurs: CLI × Notion × Coding Agent  |  Heute: Neue Notion-Datenbank per CC Prompt + Eintrag via Bot

🎯 Lernziel

Du legst eine neue Notion-Datenbank "Lernthemen" an — mit passenden Properties — und kannst neue Einträge per Telegram-Prefix Lern: hinzufügen. Nach dieser Lektion hast du eine strukturierte Sammlung aller Themen, die du noch mit dem teach-Skill lernen willst.

Warum eigene Datenbank — nicht Tagesorganizer erweitern?

Design-Entscheidung

Der Tagesorganizer ist für tägliche Tasks mit Deadline und Status. Lernthemen sind anders: kein festes Datum, kein Bereich, dafür Kategorie und teach-Workspace. Separate DB = sauberere Abfragen und kein Properties-Chaos.

Schritt 1: Parent-Seite finden

Die neue Datenbank muss unter einer Notion-Seite angelegt werden. Dein Organizer liegt auf Workspace-Level. So findest du die Page-ID:

Finde die Notion-Seite mit dem Namen "Organizer" und zeig mir ihre Page-ID.

Claude nutzt notion_find / notion_search und gibt dir die ID — sieht aus wie abc12345-6789-.... Diese ID brauchst du für Schritt 2.

Schritt 2: Datenbank anlegen — CC Prompt

Erstelle eine neue Notion-Datenbank mit dem Titel "Lernthemen" als Unterseite
der Organizer-Seite (parent_page_id: DEINE-PAGE-ID-HIER).

Properties:
- Name: title
- Status: status (Optionen: Offen, In Bearbeitung, Abgeschlossen; Standard: Offen)
- Kategorie: select (Optionen: Programmierung, Sprachen, Mathematik, Design, Sonstiges)
- Priorität: select (Optionen: Hoch, Mittel, Niedrig)
- Workspace: rich_text
- Notiz: rich_text

Zeige mir nach dem Anlegen die data_source_id der neuen Datenbank.
Wichtig nach dem Anlegen

Die data_source_id aus der Antwort merken und in CLAUDE.md eintragen — so kann Claude in Zukunft direkt auf die DB zugreifen ohne zu suchen.

Schritt 4: Bot-Prefix "Lern:" einbauen

Gleiche Technik wie Task: aus Lektion 13. Neuen System-Prompt + Routing in bot.py:

LERN_SYSTEM_PROMPT = """Du bist ein Lernthemen-Assistent. Der Nutzer nennt ein Thema, das er lernen möchte.
Lege es in der Lernthemen-Datenbank an (data_source_id: DEINE-LERNTHEMEN-DB-ID).
Leite aus dem Text ab: Name, Kategorie (Programmierung/Sprachen/Mathematik/Design/Sonstiges,
Programmierung falls unklar), Priorität (Mittel falls nicht angegeben).
Antworte NUR mit einer Zeile: 📚 Lernthema gespeichert: [Name] · [Kategorie] · [Priorität]"""

# In der Routing-Logik (nach dem Task:-Block):
elif text.lower().startswith("lern:"):
    lern_text = text[5:].strip()
    response = run_claude(lern_text, system_prompt=LERN_SYSTEM_PROMPT)
Pattern: Ein System-Prompt pro Datenbank

Jede neue Notion-DB bekommt ihren eigenen System-Prompt + Bot-Prefix. Das Muster ist immer gleich: data_source_id fest im Prompt verankern, Defaults für optionale Felder, Output-Format für Telegram.
