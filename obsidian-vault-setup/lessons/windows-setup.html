Lesson 1 / 3 · Obsidian Vault Setup

Windows: Vault-Sync + Web Clipper einrichten

Ziel: dein bestehender Obsidian-Vault ist auf diesem Windows-Gerät vollständig verfügbar, synchronisiert über das gleiche selbst-gehostete Setup wie am Desktop — inklusive Web Clipper zum Wissen-Reinspeichern aus dem Browser.

Was du brauchst

DingWoher
Obsidian Desktop-Appobsidian.md/download
Server-Adresse deines Sync-Backendsbei dir bereits vorhanden
Username / Passwort / DatenbanknamePasswortmanager oder Server-Config — nicht hier in der Lesson, aus Sicherheitsgründen
Wichtig Diese Lesson enthält bewusst keine echten Zugangsdaten (öffentlich gehostete Seite). Wo Werte gebraucht werden, steht <dein-wert> — die echten Werte hast du dir bereits notiert bzw. holst du sie aus deiner eigenen Server-Config.

Schritt für Schritt

1
Obsidian installieren & Vault öffnen Installer ausführen, Obsidian starten. Beim ersten Start: "Create new vault" oder "Open folder as vault" — der Name/Inhalt ist erstmal egal, er wird gleich durch deine echten Notizen ersetzt.
2
Community Plugins aktivieren Einstellungen (Zahnrad unten links) → Community plugins → Restricted Mode ausschalten und bestätigen.
3
Self-hosted LiveSync installieren Browse klicken → suchen: Self-hosted LiveSync → Install → Enable.
4
Plugin-Einstellungen öffnen In der linken Seitenleiste erscheint jetzt ein LiveSync-Icon (Kreis-Pfeile). Anklicken öffnet die Plugin-Einstellungen direkt.
5
Server manuell konfigurieren Zum Abschnitt Remote Database Configuration scrollen. Felder einzeln ausfüllen — URI und Datenbankname sind getrennte Felder, den Pfad nicht an die URI anhängen:
FeldWert
URIhttps://<dein-server>
Username<dein-username>
Password<dein-passwort>
Database name<deine-db>
6
Verbindung testen Button Test Database Connection klicken. Grüne Bestätigung = Server erreichbar, Login korrekt.
7
Fetch statt Rebuild Beim ersten Verbinden fragt das Plugin, wie verfahren werden soll. Fetch wählen — holt die bestehenden Notizen vom Server. Rebuild würde stattdessen den lokalen (leeren) Vault-Zustand hochladen und die Server-Daten überschreiben.
Check Nach dem Fetch: Notizanzahl in der Seitenleiste mit deinem anderen Gerät vergleichen. Stimmt sie überein, ist der Vault vollständig da.
8
Web Clipper Browser-Extension installieren Im Browser (Chrome, Firefox oder Edge) die offizielle Erweiterung Obsidian Web Clipper installieren.
9
Clipper mit Vault verbinden Extension-Icon → Einstellungen → als Ziel-Vault deinen Vault-Namen auswählen. Standardmäßig öffnet der Clipper beim Speichern kurz Obsidian über ein obsidian://-Link, das reicht ohne Zusatz-Plugin. Für nahtloses Speichern ohne Fensterwechsel: zusätzlich das Community-Plugin Local REST API installieren und in den Clipper-Einstellungen den angezeigten API-Key eintragen.

Quiz

2. Wo trägst du URI, Username, Passwort und Datenbankname ein?

3. Was ist der einfachste Weg, den Web Clipper ohne Zusatz-Plugin ans Speichern zu bekommen?

Zusammenfassung

Du hast Self-hosted LiveSync installiert, manuell mit deinem Server verbunden, den bestehenden Vault per Fetch übernommen und den Web Clipper fürs Wissen-Reinspeichern aus dem Browser eingerichtet. Windows ist damit vollständig auf dem Stand deines Desktop-Setups.
