Dependencies installieren

Ziel: alle Python-Pakete, die Voice Flow braucht, sind im venv installiert — inklusive der optionalen GPU-Beschleunigung, falls das Gerät eine NVIDIA-Grafikkarte hat.

1

Pflicht-Pakete installieren

Im Repo-Ordner, venv-Python direkt ansprechen (funktioniert egal ob aktiviert oder nicht — siehe Lektion 3):

PowerShell
PS> .venv\Scripts\python.exe -m pip install --upgrade pip PS> .venv\Scripts\python.exe -m pip install -r requirements.txt Collecting sounddevice ... Collecting faster-whisper ... Collecting numpy ... Successfully installed faster-whisper-... numpy-... pynput-... ...

Das dauert je nach Internetverbindung ein bis mehrere Minuten — faster-whisper zieht einige Abhängigkeiten nach.

2

Was installiert wird

sounddeviceMikrofon-Aufnahme
faster-whisperlokale Sprache-zu-Text-Engine (Whisper-Modell)
numpyAudio-Datenverarbeitung
pynputglobaler Hotkey-Listener (Ctrl+Alt+V)
pyperclipText ins Clipboard / Auto-Paste
pystray, PillowTray-Icon in der Taskleiste
3

Optional: GPU-Beschleunigung (nur mit NVIDIA-Grafikkarte)

Ohne diesen Schritt läuft die Spracherkennung auf CPU — funktioniert einwandfrei, nur langsamer. Mit NVIDIA-GPU zusätzlich:

PowerShell
PS> nvidia-smi (zeigt Treiber-Version + GPU, wenn Treiber korrekt installiert ist) PS> .venv\Scripts\python.exe -m pip install nvidia-cublas-cu12 nvidia-cudnn-cu12

nvidia-smi meldet Fehler statt GPU-Infos? → NVIDIA-Treiber fehlt oder ist veraltet. Aktuellen Treiber von nvidia.com/Download installieren, danach neu prüfen.

start.bat / start_silent.vbs setzen den PATH zu den DLLs dieser Pakete automatisch — kein manueller PATH-Eintrag nötig.

Warnungen beim Installieren — ignorierbar?

Zwei Meldungen tauchen später beim ersten App-Start harmlos auf und brauchen keine Aktion:

Warning: unauthenticated requests to the HF Hub — nur ein Hinweis auf langsamere Downloads ohne HuggingFace-Token, keine Fehlfunktion.

cache-system uses symlinks ... your machine does not support them — Windows braucht Admin/Entwicklermodus für Symlinks, ohne das läuft der Cache nur „degradiert" (mehr Speicherplatz, keine Fehlfunktion).

Kurzcheck

Du hast keine NVIDIA-Grafikkarte. Musst du die GPU-Pakete trotzdem installieren?

Gelernt: Pflicht-Pakete via requirements.txt, was jedes Paket tut, optionale GPU-Pakete nur bei NVIDIA-Hardware, zwei harmlose Warnungen die man ignorieren kann.
