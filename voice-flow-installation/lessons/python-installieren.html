Lektion 2 / 7

Python 3.12 installieren

Ziel: Python 3.12 ist installiert, im PATH eingetragen, und py -3.12 --version funktioniert im Terminal.

1

Installer herunterladen

Öffne python.org/downloads/release/python-3129 und lade den „Windows installer (64-bit)" herunter.

2

Installer starten — Checkbox nicht vergessen!

Doppelklick auf die heruntergeladene .exe-Datei. Es öffnet sich dieses Fenster (nachgebaut, entspricht dem echten Installer):

Wichtigster Schritt im ganzen Guide: Haken bei „Add python.exe to PATH" setzen, bevor du „Install Now" klickst. Ohne diesen Haken findet das Terminal später den Befehl python nicht — alle nachfolgenden Schritte scheitern.
3

Installation abschließen

„Install Now" klicken, warten bis der Fortschrittsbalken durch ist, „Close" klicken. Kein Neustart nötig.

4

Prüfen ob's geklappt hat

Neues Terminal öffnen (wichtig: neues Fenster, altes kennt den aktualisierten PATH noch nicht) und eintippen:

py -3.12 --version

Erwartete Ausgabe:

Python 3.12.9

Kommt stattdessen 'py' is not recognized → Haken aus Schritt 2 wurde vergessen. Installer nochmal ausführen, diesmal „Modify" wählen und PATH-Option nachträglich aktivieren.

Kurzcheck

Du hast den Installer ohne die PATH-Checkbox durchlaufen lassen. Was passiert?

Gelernt: Installer von python.org, PATH-Checkbox ist der kritische Schritt, Verifikation über py -3.12 --version in einem frisch geöffneten Terminal.
