Lektion 5 / 7

Claude CLI installieren und einloggen

Ziel: claude ist installiert und eingeloggt — das ist der Schritt, der beim ersten Laptop-Setup tatsächlich vergessen wurde und zu 401 Invalid authentication führte.

Voice Flow ruft für den Cleanup-Schritt (Text sprachlich aufräumen) im Hintergrund die claude-Kommandozeile auf. Ohne eingeloggte CLI schlägt genau dieser Schritt fehl — die Aufnahme und Transkription laufen trotzdem durch, nur der finale Text bleibt aus.

1

Node.js installieren (falls nötig)

PowerShell
PS> node --version v20.11.0

Fehler 'node' is not recognized: nodejs.org → LTS-Version herunterladen, Installer mit Standardeinstellungen durchklicken.

2

Claude CLI global installieren

PowerShell
PS> npm install -g @anthropic-ai/claude-code added 1 package in 4s
3

Einloggen — der Schritt, der leicht vergessen wird

PowerShell
PS> claude Willkommen bei Claude Code — Login erforderlich ... Öffne Browser zum Anmelden ...

Es öffnet sich der Browser mit dem Login-Fenster:

Mit dem gleichen Konto anmelden wie auf deinem Hauptgerät — dann nutzt die CLI dein bestehendes Abo, keine zusätzlichen Kosten.

4

Login testen

PowerShell
PS> claude -p "sag hallo" Hallo! Wie kann ich helfen?

Kommt stattdessen API Error: 401 Invalid authentication credentials → Login ist nicht durchgelaufen oder abgelaufen. Zurück zu Schritt 3, in der Session /login eintippen.

Kurzcheck

Voice Flow läuft, Aufnahme funktioniert, aber im Textfeld kommt nichts an und die Konsole zeigt 401 Invalid authentication. Woran liegt's?

Gelernt: Node + Claude CLI installieren, mit dem bestehenden Konto einloggen, Login mit claude -p "..." verifizieren. Das ist die häufigste Ursache für 401-Fehler beim Diktat.
