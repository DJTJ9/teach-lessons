Voraussetzungen — was du vor dem Start brauchst

Ziel dieser Lektion: du weißt genau, welche Programme auf dem neuen Gerät installiert sein müssen, bevor Voice Flow überhaupt starten kann — und warum.

Voice Flow läuft komplett lokal: Aufnahme → lokales Whisper-Modell (Transkription) → Claude-CLI (Text aufräumen) → Auto-Paste ins aktive Fenster. Kein eigener Server nötig — aber drei fremde Programme müssen auf dem Gerät installiert sein, sonst startet gar nichts oder einzelne Schritte schlagen fehl.

Die Checkliste

Python 3.12PflichtFührt den gesamten App-Code aus. Genau 3.12 — neuere Versionen (z.B. 3.14) haben oft noch keine passenden Wheels für faster-whisper/ctranslate2, dann bricht die Installation ab.
GitPflichtZum Klonen des Repos DJTJ9/voice-recorder und für die automatische Update-Funktion (git pull bei jedem Start).
Node.js + Claude CLIPflichtDer Cleanup-Schritt ruft im Hintergrund claude -p ... auf, um den diktierten Text sprachlich aufzuräumen. Ohne eingeloggte CLI kommt beim Diktat ein 401 Invalid authentication-Fehler.
NVIDIA-TreiberOptionalNur wenn eine NVIDIA-Grafikkarte im Gerät steckt und die Transkription per GPU beschleunigt werden soll. Ohne GPU läuft alles auf CPU — etwas langsamer, aber funktioniert genauso.

Warum genau diese Reihenfolge?

Jede Lektion danach baut auf der vorherigen auf:

1. Python 3.12  →  2. Repo + venv  →  3. Dependencies  →  4. Claude CLI  →  5. App starten

Überspringt man einen Schritt, merkt man das meist erst später — als kryptische Fehlermeldung mitten im Betrieb. Lektion 7 sammelt genau die drei Fehler, die bei einem echten Laptop-Setup tatsächlich aufgetreten sind, falls doch mal was schiefläuft.

Was du NICHT brauchst

Kein eigener Account bei OpenAI/Whisper (Modell läuft lokal, kein API-Call). Kein Windows-Admin-Recht für die normale Installation (nur bei manchen Edge-Cases wie Windows-Entwicklermodus für Symlinks — nicht Pflicht). Kein bezahlter Zusatzdienst — Claude CLI nutzt dein bestehendes Abo.

Kurzcheck

Warum genau Python 3.12 und nicht die neueste Version (z.B. 3.14)?

Gelernt: Vier Voraussetzungen (Python 3.12, Git, Node/Claude CLI, optional NVIDIA-Treiber), warum genau diese Python-Version, und dass alles lokal läuft ohne zusätzliche Accounts. Weiter geht's mit der Python-Installation.
