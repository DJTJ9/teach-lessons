Lektion 6 / 7

App starten und testen

Ziel: Voice Flow läuft, das Tray-Icon ist sichtbar, ein Test-Diktat kommt sauber im Textfeld an.

1

Ersten Start mit Konsole machen (zeigt Fehler)

Im Repo-Ordner (Explorer) auf desktop\start.bat doppelklicken, oder im Terminal:

PowerShell
PS> desktop\start.bat git pull ... (Auto-Update) Downloading Systran/faster-whisper-small ...

Beim allerersten Start lädt faster-whisper das Sprachmodell herunter (paar hundert MB) — braucht Internet und dauert je nach Verbindung 1–3 Minuten. Danach ist es lokal gecacht, jeder weitere Start ist sofort da.

2

Tray-Icon prüfen

Sobald der Download fertig ist, erscheint unten rechts in der Taskleiste ein lilafarbenes Kreis-Icon:

Icon nicht sichtbar? Pfeil „Ausgeblendete Symbole einblenden" (^) links neben der Uhrzeit anklicken — Windows versteckt neue Tray-Icons manchmal dort.

3

Test-Diktat

Beliebiges Textfeld anklicken (Editor, Browser, Word), dann:

  1. Strg + Alt + V gedrückt halten
  2. Ein bis zwei Sätze sprechen
  3. Tasten loslassen
  4. Kurz warten (Transkription + Cleanup) — der aufgeräumte Text erscheint automatisch im Textfeld

Passiert nichts nach dem Loslassen: Konsolenfenster von start.bat zeigt den Fehler direkt an. Lektion 7 sammelt die drei häufigsten.

4

Alltagsbetrieb ohne Konsolenfenster

Wenn der Test klappt, künftig desktop\start_silent.vbs doppelklicken statt start.bat — startet lautlos ohne sichtbares Fenster, nur das Tray-Icon zeigt den Status.

Optional: Verknüpfung zu start_silent.vbs in den Windows-Autostart-Ordner legen (Win + R → shell:startup), dann startet Voice Flow automatisch mit Windows.

Timing-Log

Jedes Diktat schreibt eine Zeile nach ~/.voice-flow/timing.log — nützlich um zu sehen, ob CPU oder GPU genutzt wird und wie lange jeder Schritt braucht:

timing.log
2026-07-22T14:03:11 record-stop=120ms stt=1800ms cleanup=900ms paste=40ms total=2860ms

Kurzcheck

Beim allerersten Start dauert es 1–3 Minuten bevor überhaupt etwas passiert. Warum?

Gelernt: Erster Start lädt das Whisper-Modell (einmalig), Tray-Icon-Check, Test-Diktat mit Strg+Alt+V, Umstieg auf start_silent.vbs für den Alltag, Timing-Log zur Diagnose.
