# Mission

## Why
Node.js ist ein Kernwerkzeug für Backend-Webentwicklung und steht in vielen Junior-Stellenausschreibungen. Ziel: selbstständig einfache APIs und Server mit Node.js bauen können.

## Success looks like
- Erklären können was Node.js ist und wofür es genutzt wird
- Einfache JavaScript-Syntax schreiben (Variablen, Funktionen, async/await)
- HTTP-Server und REST-API mit Express.js aufbauen
- npm-Pakete installieren und nutzen
- Mini-Webhook-Server als Praxisprojekt fertigstellen (Telegram-ähnlich zum eigenen Bot)

## Constraints
- Kein JavaScript-Vorwissen
- Vorwissen: Python und C# — Vergleiche als Lernhilfe
- 6+ Stunden pro Woche
- Fokus auf serverseitiges Node.js

## Out of scope
- TypeScript
- React / Next.js / Frontend-Frameworks
- Datenbanken (MongoDB, PostgreSQL)
- Docker / Deployment
- Testing in Node.js
