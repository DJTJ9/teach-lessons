# LR-0001: 3-Ebenen-Modell (Projekt / Epic / Task)

Projektplanung braucht drei Abstraktionsebenen: Projekt (Gesamtziel), Epic (Themenbereich), Task (konkrete Tagesaufgabe). Ohne Epics verwaisen Tasks ohne Kontext. Tasks müssen klein genug sein für einen Sitz (max. 4h).
