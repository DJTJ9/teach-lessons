# LR-0004: morgen-Befehl als Bot-Command

Ein dedizierter `morgen`-Befehl im Bot liefert präziseren Output als freier Freitext — weil der System-Prompt exakt filtert (Datum = heute ODER ohne Datum, Status ≠ Done) und Projekt/Epic optional inline anzeigt. Das Routing-Prinzip: spezifische Befehle vor dem allgemeinen `else`-Zweig.
