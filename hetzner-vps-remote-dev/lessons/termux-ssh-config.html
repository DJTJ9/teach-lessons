Termux + SSH Config auf Android

Terminal-Emulator für Android einrichten und SSH-Config für schnellen Verbindungsaufbau.

⚠ Termux immer aus F-Droid installieren — Play-Store-Version eingefroren seit 2020, keine Updates.

Schritt 1: F-Droid + Termux

1
f-droid.org im Browser öffnen

Alternativer App-Store für Open-Source-Apps

2
F-Droid APK herunterladen + installieren

Erlaubnis für unbekannte Quellen geben

3
In F-Droid: Termux suchen → installieren

Schritt 2: OpenSSH

pkg update && pkg install openssh

Schritt 3: SSH Key generieren

ssh-keygen -t ed25519

Enter dreimal drücken. Public Key anzeigen:

cat ~/.ssh/id_ed25519.pub

Schritt 4: Public Key auf Server eintragen

Vom PC-Terminal ausführen:

ssh root@195.201.121.96 "echo 'DEIN_PUBLIC_KEY' >> ~/.ssh/authorized_keys"

Schritt 5: SSH Config (printf statt heredoc)

⚠ Heredoc hat in Termux Zeilenumbruch-Probleme. printf verwenden:
mkdir -p ~/.ssh
printf 'Host dev\n    HostName 100.64.150.74\n    User root\n    IdentityFile ~/.ssh/id_ed25519\n' >> ~/.ssh/config

Schritt 6: Testen

ssh dev

Zusammenfassung

  • Termux aus F-Droid — Play-Store-Version veraltet

  • printf statt heredoc in Termux
  • Nach Setup: ssh dev reicht
