Server-Grundinstallation

System-Updates, Git, Node.js und Claude Code CLI installieren.

Schritt 1: System aktualisieren

apt update && apt upgrade -y

Frischer Server hat veraltete Pakete. Updates schließen Sicherheitslücken und stellen Kompatibilität sicher.

Schritt 2: Basis-Tools

apt install -y curl git wget unzip zip build-essential
PaketWozu
curlURLs herunterladen
gitVersionskontrolle
wgetDateien herunterladen
unzipZIP entpacken
build-essentialC/C++ Compiler für native Node-Module

Schritt 3: NVM installieren

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

NVM erlaubt mehrere Node-Versionen parallel, kein Root für Updates nötig.

Schritt 4: Node.js + Claude Code

source ~/.bashrc && nvm install --lts && npm install -g @anthropic-ai/claude-code

source ~/.bashrc lädt NVM-Pfad in die aktuelle Shell — ohne diesen Befehl ist nvm nicht gefunden.

⚠ Falls "command not found" nach Installation: source ~/.bashrc ausführen.

Quiz

Warum source ~/.bashrc nach NVM?

Weil .bashrc NVM-Passwort enthält
Aktuelle Shell kennt neue NVM-Pfad-Einträge noch nicht
Weil Node.js neugestartet wird

Zusammenfassung

  • apt update + upgrade: System aktuell halten
  • NVM: flexibler als direktes Node.js
  • source ~/.bashrc: NVM-Pfad in Shell laden
  • npm install -g: Claude Code global installieren
