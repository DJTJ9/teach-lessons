Neues Gerät hinzufügen

Schritt für Schritt ein neues Gerät mit deinem Server verbinden. Alle Werte sind deine persönlichen Serverdaten.

ℹ Deine Daten: Public IP 195.201.121.96 · Tailscale IP 100.64.150.74 · SSH Alias ssh dev · User root

Übersicht

  1. SSH Key auf neuem Gerät generieren
  2. Public Key auf Server eintragen
  3. Tailscale installieren + verbinden
  4. SSH Config einrichten
  5. Verbindung testen

Schritt 1: SSH Key generieren

PC/Laptop:

ssh-keygen -t ed25519 -C "neues-geraet"

Android (Termux):

pkg install openssh
ssh-keygen -t ed25519

Public Key anzeigen:

cat ~/.ssh/id_ed25519.pub

Schritt 2: Public Key auf Server eintragen

Von einem Gerät mit bestehendem Zugriff:

ssh root@195.201.121.96 "echo 'HIER_PUBLIC_KEY' >> ~/.ssh/authorized_keys"
⚠ Kompletten Output von cat ~/.ssh/id_ed25519.pub des neuen Geräts einsetzen.

Schritt 3: Tailscale installieren

Windows/Mac: tailscale.com/download → Installer → mit Account einloggen

Android: Play Store → Tailscale → mit Account einloggen

Server-Tailscale-IP bleibt immer 100.64.150.74.

Schritt 4: SSH Config

PC (PowerShell):

Add-Content "$env:USERPROFILE\.ssh\config" "`nHost dev`n    HostName 100.64.150.74`n    User root`n    IdentityFile ~/.ssh/id_ed25519"

Android (Termux):

mkdir -p ~/.ssh
printf 'Host dev\n    HostName 100.64.150.74\n    User root\n    IdentityFile ~/.ssh/id_ed25519\n' >> ~/.ssh/config

Schritt 5: Verbindung testen

ssh dev

Erwartet: root@dev:~#

Schritt 6: Claude Code

claude

Ersten Start: Login with Claude.ai → Link im Browser → autorisieren.

Checkliste neues Gerät

  • SSH Key generiert: ssh-keygen -t ed25519
  • Public Key auf 195.201.121.96 eingetragen
  • Tailscale installiert + Account verbunden
  • SSH Config: Host dev, 100.64.150.74
  • ssh dev getestet → root@dev Prompt
  • claude gestartet + eingeloggt
