Hetzner Account + Server erstellen

Hetzner-Account erstellen, Zahlung einrichten und ersten Server starten.

Zahlungsmittel: Revolut virtuelle Karte

Hetzner akzeptiert Kreditkarte oder SEPA. Virtuelle Revolut Debitkarte (kostenlos) funktioniert.

1
Revolut App → Konto erstellen

Identität verifizieren: Ausweis + Selfie, ca. 10 Min

2
Virtuelle Karte erstellen

Karten → + Karte hinzufügen → Virtuelle Karte

3
Kartendetails anzeigen

Auge-Symbol → Face ID/PIN → Kartennummer, Ablaufdatum, CVV

⚠ Hetzner bucht ca. €1 zur Verifikation — wird automatisch zurückgebucht.

Hetzner Server erstellen

1
hetzner.com → Cloud → Jetzt starten
2
New Project → Name: dev-server
3
+ Create Server
4
Location: Nürnberg oder Falkenstein

EU-Rechenzentrum, niedrige Latenz

5
Image: Ubuntu 24.04

Aktueller LTS — nicht 22.04

6
Type: Shared vCPU → x86 → CX33

4 vCPUs, 8 GB RAM, ca. €10.10/Mo

7
SSH Key einfügen

Auf PC: cat ~/.ssh/id_ed25519.pub → Output in Hetzner einfügen

8
Create & Buy Now

Server läuft in Sekunden, Public IP erscheint sofort

💡 Volumes, Shielded Instance, Confidential Computing: alle überspringen.

Quiz

Warum Ubuntu 24.04 statt 22.04?

22.04 ist bei Hetzner nicht mehr verfügbar
24.04 ist der aktuellere LTS — neuer und länger supportet
24.04 ist günstiger

Zusammenfassung

  • Revolut virtuelle Debitkarte akzeptiert bei Hetzner
  • Server: Ubuntu 24.04, CX33, Nürnberg/Falkenstein
  • Extras (Volumes etc.) überspringen
  • Public IP nach Create sofort verfügbar
