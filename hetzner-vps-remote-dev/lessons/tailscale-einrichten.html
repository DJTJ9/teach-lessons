Tailscale einrichten

Tailscale verbindet deine Geräte sicher — ohne offene Ports, ohne Portweiterleitung.

Was ist Tailscale?

Privates Mesh-Netzwerk zwischen deinen Geräten. PC, Handy und Server bekommen je eine feste 100.x.x.x IP und erreichen sich direkt — egal in welchem Netzwerk. Basiert auf WireGuard.

Setup

1
tailscale.com → Account erstellen (kostenlos)
2
PC: tailscale.com/download → installieren
3
Server: 
curl -fsSL https://tailscale.com/install.sh | sh && tailscale up

Link erscheint → im Browser autorisieren

4
Android: Play Store → Tailscale → einloggen

Tailscale-IP des Servers

tailscale ip -4

Gibt 100.64.150.74 aus — diese IP wird in der SSH Config verwendet.

Tailscale-IP ist nur für deine Geräte sichtbar — sicherer als öffentliche Hetzner-IP, keine Ports öffnen nötig.
