Telegram Callback-Handler erweitern

Ziel: Wie callback_data-Routing funktioniert und wie ein neuer Button-Typ (dev_status:) neben bestehenden Handlern eingebaut wird.

Ablauf: Inline-Button → Handler

  1. Bot sendet Nachricht mit InlineKeyboardMarkup — jeder Button hat callback_data
  2. User klickt → Telegram sendet callback_query Update an Bot
  3. Bot liest callback_query["data"] → routet zu passendem Handler
  4. Bot ruft answerCallbackQuery auf (verhindert "Laden"-Animation)

Button senden

keyboard = [
    [
        {"text": "📊 Status",     "callback_data": f"proj_status:{slug}"},
        {"text": "📋 Dev Status", "callback_data": f"dev_status:{slug}"},
    ],
]
send_message(TOKEN, CHAT_ID, f"Projekt: {slug}", keyboard=keyboard)

Format: "prefix:payload" — Prefix identifiziert Handler, Payload übergibt Daten.

Handler registrieren

def _handle_callback(callback_query):
    data = callback_query["data"]

    if data.startswith("proj_status:"):
        slug = data[12:]
        _run_status(slug)          # Claude-Subprocess, ~5–30s

    elif data.startswith("dev_status:"):
        slug = data[11:]           # len("dev_status:") == 11
        send_message(TOKEN, CHAT_ID, _format_dev_status(slug))  # sofort

File-Read statt Subprocess

def _format_dev_status(slug: str) -> str:
    status_path = HUB_DIR / "topics" / slug / "STATUS.md"
    if not status_path.exists():
        return f"Kein STATUS.md fuer {slug}"

    lines = status_path.read_text().splitlines()
    fields, roadmap, in_roadmap = {}, [], False
    for line in lines:
        if line.startswith("## Roadmap"):
            in_roadmap = True
        elif in_roadmap and line.startswith("- "):
            roadmap.append(line)
        elif ":" in line and not in_roadmap:
            k, _, v = line.partition(":")
            fields[k.strip()] = v.strip()

    active = fields.get("Active", "-")
    done = sum(1 for r in roadmap if "[done]" in r)
    return f"Aktiv: {active}\nRoadmap: {done} done"

Quiz

Frage: data = "dev_status:my-project" → data[11:] — Ergebnis?

Antwort

"my-project" — len("dev_status:") ist 11.

Aufgabe: Neuer Button "🗒 Backlog" mit callback_data=f"backlog:{slug}". Welche 2 Stellen?

Antwort
  1. Button im keyboard dict hinzufügen
  2. elif data.startswith("backlog:"): in _handle_callback()

Zusammenfassung

  • Format: "prefix:payload" — Prefix → Handler, Payload → Daten
  • Neuer Button = 2 Stellen: keyboard + elif in _handle_callback()
  • File-Read-Handler: sofort, kein Subprocess für einfache Daten
  • data[len("prefix:"):] zum Payload extrahieren
